# Favicon Setup Instructions

## Generate Favicons

1. Open `generate-favicon.html` in your browser
2. You'll see a preview of the HQ logo
3. Click the download buttons to generate different sizes:
   - 512x512 (for high-res displays and app icons)
   - 256x256 
   - 128x128
   - 64x64
   - 32x32 (standard favicon)
   - 16x16 (small favicon)

4. Save all downloaded files to `/public/assets/` folder

## Files needed in `/public/assets/`:
- `favicon-512.png`
- `favicon-256.png`
- `favicon-128.png`
- `favicon-64.png`
- `favicon-32.png`
- `favicon-16.png`

The HTML is already configured to use these icons!

## Quick command to open the generator:
```bash
open generate-favicon.html
```

Or just drag `generate-favicon.html` into your browser.
